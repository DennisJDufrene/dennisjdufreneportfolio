    About Me¶

    Overview¶

    I am a Senior/Principle Technical Writer with broad experience in creating and maintaining large document repositories, documentation processes, and documentation maturity models that result in easily accessible documentation for both business and development users. This broad experience includes:

    • Technical Writing/Editing:
      Writing and editing technical documents including user manuals, training manuals, configuration manuals, video tutorials, API documentation, and context sensitive help files.
    • Docs-as-Code: Implementing and using Docs-as-Code processes and technology stacks to create technical documentation.
    • Interviewing SMEs:
      Interviewing subject-matter experts (SMEs) to gather information.
    • Researching Information:
      Researching and identifying relevant data for the documentation process.
    • Implementing Style Guides:
      Working knowledge of Chicago Manual of Style, AP Stylebook, and Microsoft Writing Style Guide.
    • Implementing Processes:
      Developing and implementing documentation processes to ensure documentation is current and correct.
